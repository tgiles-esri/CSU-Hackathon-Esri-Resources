# CSU-Hackathon-Esri-Resources
A collection of code snippets and general geospatial resources to support the CSU 2026 Hackathon

## Most examples and resources created for the Hackathon can be found in the StoryMap collection below.  The collection contains 4 sections, or stories covering a variety of topics:
- GIS Enablement:  A crash cource in GIS concepts, focusing on how GIS data and functions can be utilized via APIs and the ArcGIS API for Python.  This will be covered at the techincial enablement session prior to the start of the hackathon
- ArcGIS API for Python:  An introduction to the ArcGIS API for Python with samples of common data access and analysis patterns.  The snippets in this story are also available in the CSU 2026 Hackathon ArcGIS Python examples notebook in this repo.
- ArcGIS Online Primer:  An introduction to configurable applications that may be useful for sharing your team's results.
- ArcGIS Resources:  General Esri resources around GIS data and analysis.  Additional resource docs specific to each challenge question will be added once the Hacakthon officially begins.
